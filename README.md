### About

Computer Science graduate, currently pursuing a Master's in **Artificial Intelligence**. I work as a **Fullstack Developer** at [Altrama Italia](https://www.altrama.com), building web and mobile products end-to-end.

My real passion is **Agentic AI** — designing and deploying autonomous systems powered by LLMs. I study multi-agent architectures, LLM orchestration frameworks, tool use, and the emerging MCP ecosystem. My goal is to build agents that don't just respond, but *act*.

---

### 🤖 Agentic AI

- **Multi-agent systems** — orchestration, delegation and coordination between autonomous agents
- **Agentic frameworks** — OpenAI Agents SDK · CrewAI · LangGraph · AutoGen
- **Model Context Protocol (MCP)** — building and integrating MCP servers and tools
- **Agentic design patterns** — proven patterns for connecting and chaining LLMs effectively
- **Tool use & function calling** — extending LLMs with real-world capabilities to act autonomously
